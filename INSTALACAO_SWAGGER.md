# Instalação do Swagger

O Swagger foi adicionado ao projeto, mas precisa ser instalado.

## Instalação

Execute na raiz do projeto:

```bash
npm install
```

Ou especificamente no backend:

```bash
cd apps/backend
npm install @nestjs/swagger
```

## Acesso

Após instalar e rodar o backend, acesse:

**http://localhost:3001/api**

Você verá a documentação interativa da API com todos os endpoints, parâmetros e exemplos.
